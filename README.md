# Movies-DB-2 (Фильмотека)

Пэт проект по разработке приложения на пайтоне, для систематизации домашней фильмотеки.

Приложение сканирует указанный путь, занося все видео файлы в базу данных. В последствии можно сопоставить файлы со страницами из IMDb. По запросу пользователя приложение обращается к IMDb и если потенциальных соответствий больше чем одно, предлагает выбрать наиболее подходящее. Таким образом всякому файлу может быть сопоставлена одна страница, откуда в базу данных извлекаются подробности (дата премьеры, теги по жанрам, режиссёр, ведущие актёры и т.п.). 

Конечным продуктом должно стать десктопное приложение, которым сможет оперировать даже условная бабушка.
